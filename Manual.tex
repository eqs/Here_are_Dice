	\section{どんなゲーム？}
		このゲームは，全国高等専門学校 第 23 回プログラミングコンテストの競技部門のオマージュであり，
		「プロコン競技部門の楽しさを皆さんに知ってもらいたい！！！！」という作者の願いの下制作されました．
	\section{遊び方}
		\begin{itemize}
			\item{}操作はすべてマウスを使った操作になります．
			\item{}制限時間内に，テーブルの上に積まれた大・中・小の3種類のサイコロそれぞれの個数を数えてそれを入力します．
				正解との誤差がより小さくなるように頑張りましょう．
			\item{}競技中に，ヒントとしてテーブル上のサイコロの総重量が与えられます．
				サイコロ1つあたりの重さは，大サイコロが5.6g，中サイコロが1.3g，小サイコロが0.3gです．
		\end{itemize}
	\section{開発環境・期間}
		\begin{itemize}
			\item{}開発環境：Unity Version 5.3.4 f1，Blender 2.70a
			\item{}開発期間：企画・モデリング・コーディングあわせて1週間ぐらい．寝たい．
		\end{itemize}
